# AI 图像工具开发与单价方案

更新时间：2026-06-05

## 目标

在现有 `AI工具` 页面基础上，优先落地两个能快速变现的视觉工具：

- 图片去水印：低价、高频、适合广告解锁。
- AI 生图：通过模板降低用户输入门槛，通过模型分层做差异化收费。

产品定位不是做一个“大而全绘画站”，而是做微信里轻量可用的图片生产工具：用户上传或输入一句话，快速得到可发朋友圈、公众号、小红书、商品图、头像、教程封面的结果。

## 外部依据

- 腾讯混元生图官方产品页显示，混元生图覆盖 3.0、2.0、极速版、图像风格化、商品背景生成、线稿生图等能力；标价口径为混元 3.0 约 0.2 元/张起，2.0 约 0.1 元/张起，极速版/风格化/商品背景约 0.06 元/张起。
- OpenAI 官方价格页显示，GPT-Image-2 按 token 计费，图像输入 $8/百万 token、图像输出 $30/百万 token，文本输入 $5/百万 token；并说明图片会转换为 token 后计费。
- Google Gemini API 官方价格页显示，Gemini 3.1 Flash Image 标准档约 $0.067/1K 图、$0.101/2K 图、$0.151/4K 图，Batch 约为一半价格。
- EvoLinkAI/awesome-gpt-image-2-API-and-Prompts 是一个面向 GPT-Image-2 的提示词和案例集合，GitHub 页面显示约 16k stars、CC0-1.0 license。适合参考其模板分类与提示词结构，但不要直接把全量内容硬塞进小程序，应做成自己的场景模板库。

## 核心产品设计

### 1. 图片去水印

功能定义：

- 用户上传图片。
- 选择去除范围：自动识别 / 手动涂抹 / 框选区域。
- 后端调用腾讯图像编辑能力完成局部修复。
- 输出图片支持预览、保存、再次编辑。

第一版建议：

- 只做单图处理。
- 图片最大 5MB，前端压缩到长边 1600px。
- 默认走腾讯模型。
- 每人每天免费 1 次，超出看激励广告继续使用。
- 只保留最近 10 条历史，避免云存储成本膨胀。

合规边界：

- 页面显著提示：仅用于处理本人拥有版权或已获授权的图片。
- 禁止用于移除第三方版权标识、平台水印、摄影师署名等侵权场景。
- 对结果加“AI 处理”标识或在结果详情里展示 AI 生成/处理说明，满足深度合成审核要求。

### 2. AI 生图

功能定义：

- 用户选择模板，输入少量变量。
- 系统根据模板拼装 prompt。
- 用户选择模型档位：腾讯 / 豆包 / Google / ChatGPT。
- 后端提交任务，轮询结果，写入作品历史。

模板体系参考 EvoLinkAI 仓库的思路，但产品上要更接地气：

| 模板分类 | 示例模板 | 目标用户 | 推荐模型 |
| --- | --- | --- | --- |
| 社交头像 | 质感头像、卡通头像、职业头像 | 普通用户 | 腾讯极速版 |
| 小红书封面 | 探店封面、知识卡片、好物封面 | 内容创作者 | 腾讯 2.0 / 豆包 |
| 公众号头图 | 科技资讯头图、教程封面 | 当前小程序运营 | 腾讯 2.0 / Google |
| 商品图 | 商品背景、详情页主图 | 电商/微商 | 腾讯商品背景 |
| 表情包 | 静态表情、梗图 | 社交传播 | 腾讯极速版 |
| 高级创意 | 海报、电影感、强文字排版 | 高价值用户 | Google / GPT-Image-2 |

模板字段建议：

```json
{
  "id": "wechat_article_cover_ai_news",
  "name": "AI资讯公众号头图",
  "category": "公众号头图",
  "modelTier": "standard",
  "variables": [
    { "key": "topic", "label": "主题", "required": true },
    { "key": "style", "label": "风格", "default": "科技感、克制、干净" },
    { "key": "ratio", "label": "比例", "default": "16:9" }
  ],
  "prompt": "为一篇关于{{topic}}的AI资讯文章生成公众号头图，风格{{style}}，画面简洁高级，不要出现真实品牌Logo，不要生成可识别真人。"
}
```

## 模型路由

### 免费/广告层

使用腾讯为主：

- 图片去水印：腾讯图像编辑。
- AI 生图：腾讯极速版或混元 2.0。
- 理由：中文体验好、成本低、微信生态接入顺、成长计划额度可以覆盖冷启动。

收费策略：

- 每天 1 次免费。
- 之后看一次激励广告获得 1 次腾讯生图或去水印机会。
- 会员用户每天赠送固定次数，比如 20 次腾讯工具额度。

### 普通付费层

腾讯 2.0 / 豆包：

- 用于小红书封面、公众号头图、商品图等更偏实用的图片。
- 单张售价建议：0.99 元/张。
- 会员价：0.5 元/张或包月赠送 60 张。

### 高级付费层

Google / GPT-Image-2：

- 用于复杂文字排版、高质量海报、商业头图、强审美需求。
- 单张售价建议：
  - Google：2.99 元/张。
  - GPT-Image-2：4.99 元/张起。
  - 高清/多图方案：9.9 元生成 3 张。
- 会员价：
  - GO 会员：Google 2.49 元/张，GPT 4.49 元/张。
  - PLUS 会员：Google 1.99 元/张，GPT 3.99 元/张。

注意：Google 和 OpenAI 都要设置日预算、单用户限流和失败重试上限。不要让高级模型走无限会员，否则很容易亏损。

## 推荐单价表

| 功能 | 模型 | 成本粗估 | 用户价格 | 获取方式 |
| --- | --- | ---: | ---: | --- |
| 图片去水印 | 腾讯 | 低，按图像处理/生图能力计 | 免费/广告 | 每天 1 次免费，之后看广告 |
| 腾讯快速生图 | 腾讯极速版 | 约 0.06 元/张起 | 免费/广告 | 每天 1 次免费，之后看广告 |
| 标准生图 | 腾讯 2.0 / 豆包 | 约 0.1 元/张起 | 0.99 元/张 | 会员可折扣 |
| 高级生图 | 腾讯 3.0 | 约 0.2 元/张起 | 1.99 元/张 | 会员可折扣 |
| Google 生图 | Gemini Flash Image | 约 $0.067/1K 图起 | 2.99 元/张 | 单张付费/会员折扣 |
| GPT 生图 | GPT-Image-2 | token 计费，输出成本较高 | 4.99 元/张起 | 单张付费/会员折扣 |

定价原则：

- 广告层覆盖低成本腾讯能力，核心目标是拉活和留存。
- 付费层不要按真实成本只加一点毛利，而要按“用户感知价值”定价。
- 高级模型按张收费，不建议开放无限使用。
- 会员不是无限高级模型，而是“免费次数 + 折扣价 + 排队优先 + 历史作品保存”。

## 会员权益建议

GO 会员：

- 每天 20 次腾讯工具额度。
- 标准生图 0.5 元/张。
- Google/GPT 高级模型 9 折。
- 历史作品保存 30 天。

PLUS 会员：

- 每天 60 次腾讯工具额度。
- 标准生图免费额度每月 100 张。
- Google/GPT 高级模型 8 折。
- 历史作品保存 180 天。
- 任务优先队列。

## 页面与流程

### AI 工具列表页

- 图片去水印：可用，标记“免费/广告”。
- AI 生图：可用，标记“模板生图”。
- 高级模型角标：Google / GPT。

### 图片去水印页

流程：

1. 上传图片。
2. 选择自动识别或手动框选。
3. 点击“开始处理”。
4. 如果今日免费次数已用完：弹出看广告。
5. 生成结果页展示对比图、保存、再次处理。

### AI 生图页

流程：

1. 选择模板。
2. 填写变量。
3. 选择模型档位。
4. 展示预计消耗：免费/广告/0.99/2.99/4.99。
5. 提交任务。
6. 结果页展示 1-4 张图，支持保存、复制提示词、再次生成。

## 开发拆分

### 第一期：腾讯去水印 + 腾讯模板生图

目标：7-10 天上线。

- 前端：新增图片去水印工具页。
- 前端：新增 AI 生图模板页。
- 后端：新增 `run-ai-image-tool` 云函数。
- 后端：新增工具调用记录表。
- 后端：接入腾讯图像能力。
- 计费：每日 1 次免费 + 激励广告解锁。
- 合规：增加 AI 生成/AI 处理说明。

### 第二期：付费单张和会员额度

目标：5-7 天。

- 新增工具订单表。
- 接入虚拟支付购买“图片点数”或“高级模型次数”。
- 会员中心展示剩余额度。
- 工具页展示价格和会员折扣。

### 第三期：Google / GPT-Image-2 高级模型

目标：7-14 天。

- 后端增加模型路由。
- 增加 Google/OpenAI API Key 配置。
- 增加预算保护：单用户每日上限、全局每日上限、失败重试上限。
- 增加模板适配：同一模板根据模型生成不同 prompt。

## 风险与控制

- 成本风险：高级模型必须按张收费，且设置预算阈值。
- 审核风险：深度合成类目、AI生成标识、内容安全检查必须补齐。
- 版权风险：去水印必须限制为自有/授权图片，并保留用户确认。
- 体验风险：图片任务可能慢，必须异步任务 + 轮询 + 状态页。
- 存储风险：输出图片定期清理，会员延长保存期。

## 建议结论

先做：

1. 图片去水印：腾讯模型 + 每天 1 次免费 + 看广告继续用。
2. AI 生图：腾讯模板生图 + 每天 1 次免费 + 看广告继续用。
3. 模板库先做 12 个高频模板：头像、小红书封面、公众号头图、商品背景、表情包、科技资讯头图。

暂缓：

- 一开始就接 Google/GPT 高级模型。
- 一开始就做无限会员。
- 一开始就做复杂画布涂抹。

等腾讯版有真实使用量后，再开放 Google/GPT 高级档按张付费。这样现金流和成本风险都更可控。
